<Section icon={Sparkles} title="グラフィックスとパフォーマンスの最適化">

**パワーウォッシュシミュレーター2の最適設定**で完璧なバランスを実現するには、ゲームがリアルタイムの汚れの除去や水粒子をどのように処理しているかを理解する必要があります。38のキャリアミッションや、スター・ウォーズ パックのミレニアム・ファルコンのような広大な複数ステージ環境が含まれているため、お使いのハードウェアはフレームレートを落とさずに複雑な物理演算を処理する必要があります。

PCプレイヤーの場合、目標は「汚れのハイライト（Dirt Highlight）」をはっきりと視認できるようにしつつ、60以上のFPSを維持することです。Nintendo Switch 2などのコンソール版では、精密な清掃に必要なスムーズなカメラワークを実現するために「パフォーマンスモード」を優先することが不可欠です。

| 設定項目 | 推奨値 | パフォーマンスへの影響 |
| :--- | :--- | :--- |
| **解像度スケール** | 100% | 鮮明度とFPSに大きな影響 |
| **水質** | 中〜高 | パーティクル物理に大きな影響 |
| **影の品質** | 中 | ライティングに中程度の影響 |
| **テクスチャの詳細度** | 高 | 影響は軽微（VRAM依存） |
| **アンチエイリアス** | TAAまたはDLSS | エッジの滑らかさに中程度の影響 |
| **汚れの残留表示** | 高 | 視覚的フィードバックに中程度の影響 |
| **フレームレート制限** | 無制限 / 60 FPS | GPU温度に大きな影響 |

<Callout type="warning" title="VRAMの管理">
ショッピングモールや18輪トレーラー（18 Wheeler）などの大規模な依頼でカクつき（スタッター）が発生する場合は、まず**影の品質**と**解像度スケール**を下げてください。画面が水粒子や泡で満たされる際、これらの設定が最もリソースを消費します。
</Callout>

<Section icon={Target} title="入力と操作設定">

高圧洗浄機を何時間も操作するのは体に負担がかかります。本作では、コミュニティから強く要望されていたQOL（快適性）向上のためのいくつかの「設定」が導入されました。入力感度やボタン割り当てを調整することで、1つの車両や建物の清掃にかかる時間を大幅に短縮できます。

<Steps>
  <Step num="1" title="「ウォッシュ切り替え」を有効にする">
    オプションメニューを開き、「ウォッシュモード（Wash Mode）」設定を探します。これを「ホールド（Hold）」から「トグル（Toggle）」に変更します。これにより、トリガーやマウスボタンを押し続けなくても継続的に放水できるようになり、38の依頼があるキャリアキャンペーンを攻略する上で非常に重要になります。
  </Step>
  <Step num="2" title="エイム感度を調整する">
    パワーウォッシュシミュレーター2では、広範囲の洗浄とピクセル単位の精密な作業の両方が求められます。大きなパネルの清掃用に「通常感度（General Sensitivity）」を高めに設定し、継ぎ目、ボルト、小さなトリムの清掃用に「精密エイム感度（Precision Aim Sensitivity）」を低めに設定しておきましょう。
  </Step>
  <Step num="3" title="視野角（FOV）を最適化する">
    FOV（視野角）を少なくとも90以上に設定します。視野が広がると、飛行船や寺院の内部といった巨大な建造物の清掃状況を把握しやすくなり、画面酔いの原因となる「トンネル視界」効果を大幅に軽減できます。
  </Step>
  <Step num="4" title="ノズルホイールを設定する">
    ノズルの回転や選択キーが、アクセスしやすいボタンに割り当てられているか確認してください。スター・ウォーズ パックのミッションなど、一刻を争う状況では、25°ノズルから0°の赤色ノズルへ素早く切り替えることが効率化の鍵となります。
  </Step>
</Steps>

<Callout type="tip" title="プロコントローラーの設定">
コンソール版のプレイヤーには、「汚れのハイライト（Dirt Highlight）」をパドル（背面ボタン）や押しやすいショルダーボタン（L/Rボタンなど）に割り当てることを強くお勧めします。最後のわずかな汚れを見つけるために、このボタンを何千回も連打することになります。
</Callout>

</Section>

<Section icon={Shield} title="マルチプレイヤーと接続の最適化">

パワーウォッシュシミュレーター2は、オンラインで最大4人、ローカル分割画面で2人プレイに対応しています。特にXboxとPCなど、異なるプラットフォーム間でプレイする際にラグのない快適な環境を確保するには、ネットワーク「設定」を正しく構成する必要があります。

**マルチプレイヤー接続のヒント:**
- **クロスプレイ:** 異なるエコシステム（Xbox/PCなど）のフレンドと遊ぶ場合は、ゲームプレイ設定で「クロスプラットフォームプレイ（Cross-Platform Play）」が「オン」になっていることを確認してください。
- **セッションのプライバシー:** 意図しないプレイヤーが参加して特定の清掃順序を乱すのを防ぐため、セッションを「フレンドのみ（Friends Only）」に設定するか、「ルームコード」を使用してください。
- **進行状況の共有:** キャリアの進行状況はホストのみが保存されますが、参加したすべてのプレイヤーは自分のホームベースの家具に使用できる「パワーウォッシュポイント」を獲得できます。
