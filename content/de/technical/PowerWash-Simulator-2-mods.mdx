<Section icon={Map} title="Installationsanleitung: So installieren Sie Mods sicher">

Das Einrichten von Mods für PowerWash Simulator 2 ist unkompliziert, sofern Sie einen seriösen Mod-Manager oder eine Trainer-Engine verwenden. Befolgen Sie diese Schritte, um eine saubere Installation zu gewährleisten, ohne Ihre Speicherstände zu riskieren.

<Steps>
  <Step num="1" title="Sichern Sie Ihre Speicherdaten">
    Bevor Sie **PowerWash Simulator 2 Mods** installieren, navigieren Sie zu Ihrem lokalen AppData-Ordner und kopieren Sie Ihre Speicherdateien an einen sicheren Ort. Modding kann gelegentlich Synchronisierungsprobleme mit Cloud-Speichern verursachen.
  </Step>
  <Step num="2" title="Laden Sie einen vertrauenswürdigen Trainer herunter">
    Verwenden Sie eine verifizierte Plattform wie WeMod oder eine ähnliche Trainer-Engine. Diese Plattformen erkennen die Spielversion automatisch und wenden die notwendigen Hooks an, ohne dass Dateien manuell verschoben werden müssen.
  </Step>
  <Step num="3" title="Starten Sie das Spiel über die Mod-Engine">
    Um sicherzustellen, dass die Mods korrekt injiziert werden, starten Sie das Spiel immer über die Benutzeroberfläche des Trainers und nicht über den Standard-Steam- oder Epic Games-Launcher.
  </Step>
  <Step num="4" title="Hotkeys zuweisen">
    Belegen Sie Ihre Lieblings-Mods (wie Credits hinzufügen oder unendlicher Druck) mit den Tasten F1-F12. So können Sie sie ein- und ausschalten, ohne das Spiel mit Alt-Tab verlassen zu müssen.
  </Step>
</Steps>

| Installationsschritt | Schwierigkeit | Benötigte Zeit |
| :--- | :--- | :--- |
| Speicher-Backup | Einfach | 2 Minuten |
| Trainer-Setup | Mittel | 5 Minuten |
| Funktions-Mapping | Einfach | 3 Minuten |
| Spielstart | Einfach | 1 Minute |

<Callout type="success" title="Erfolg">
Sobald der Trainer "Spielt" anzeigt, können Sie Ihre Hotkeys im Spiel verwenden. Sie sollten einen Signalton hören, der bestätigt, dass die Mod aktiv ist.
</Callout>

</Section>
