<Section icon={Settings} title="So aktivieren und richten Sie Multiplayer-Sitzungen ein">

Das Einrichten einer Sitzung für **PowerWash Simulator 2 Crossplay** ist ein unkomplizierter Prozess, erfordert jedoch, dass der Host die Online-Funktionen manuell aktiviert. Standardmäßig starten viele Sitzungen im „Solo“-Modus, um die Leistung zu schonen und unerwünschte Unterbrechungen zu verhindern. Um Freunde in Ihre Welt zu holen, müssen Sie zum multifunktionalen Sozial-Menü navigieren, das sich im Tablet-Interface oder im Haupt-Pausebildschirm befindet.

<Steps>
  <Step num="1" title="Sozial-Menü aufrufen">
    Starten Sie Ihren gewünschten Reinigungsauftrag im Karriere- oder Freien Spielmodus. Sobald das Level geladen ist, drücken Sie die Escape-Taste (PC) oder die Optionen/Menü-Taste (Konsole). Suchen Sie nach dem Symbol, das zwei nebeneinander stehende Personen darstellt.
  </Step>
  <Step num="2" title="Online-Multiplayer umschalten">
    In der oberen rechten Ecke des Sozial-Overlays sehen Sie einen Schalter für „Online-Multiplayer“. Stellen Sie diesen auf die Position „Ein“. Diese Aktion initialisiert den Spielserver für Ihre spezifische Sitzung.
  </Step>
  <Step num="3" title="Raum-Code generieren und kopieren">
    Sobald Sie online sind, erscheint ein Feld „Raum-Code“. Klicken Sie auf die Schaltfläche „Kopieren“ neben dem Code. Diese eindeutige alphanumerische Zeichenfolge benötigen Ihre Freunde auf anderen Plattformen, um Ihrem Spiel beizutreten.
  </Step>
  <Step num="4" title="Code teilen">
    Senden Sie den kopierten Code über Discord, Textnachricht oder plattformspezifische Messaging-Dienste an Ihre Freunde. Diese müssen den Code im Bereich „Einem Spiel beitreten“ ihres eigenen Hauptmenüs eingeben, um sich mit Ihrer Instanz zu verbinden.
  </Step>
</Steps>

<Callout type="warning" title="Sichtbarkeit der Sitzung">
Die Aktivierung von **PowerWash Simulator 2 Crossplay** über Raum-Codes macht Ihr Spiel nicht für die ganze Welt öffentlich. Nur Spieler mit Ihrem spezifischen Code können beitreten, wodurch sichergestellt wird, dass Ihre privaten Reinigungssitzungen vor zufälligen Spielern geschützt bleiben.
</Callout>

<Section icon={Users} title="Kooperative Spielmodi und Limits">

Multiplayer in PowerWash Simulator 2 bedeutet nicht nur Gesellschaft; es geht um Effizienz. Das Spiel skaliert seine Belohnungen und Herausforderungen basierend auf der Anzahl der aktiven Reiniger. Es gibt jedoch spezifische Einschränkungen, wie viele Spieler eine einzelne „Waschzone“ besetzen können, abhängig vom gewählten Modus. Das Verständnis dieser Einschränkungen hilft bei der Planung Ihres Reinigungsplans für 2026 mit Ihrer Crew.

<CardGrid cols={3}>
  <Card title="Karriere-Modus">
    - **Host-zentriert**: Nur der Host speichert den Auftragsfortschritt.
    - **Spielerlimit**: Bis zu 6 Spieler auf großen Karten.
    - **Wirtschaft**: Gäste verdienen teilweise Währung für Upgrades.
  </Card>
  <Card title="Freies Spiel">
    - **Lockerere Regeln**: Jede freigeschaltete Karte kann gespielt werden.
    - **Spielerlimit**: Bis zu 6 Spieler, unabhängig von der Kartengröße.
    - **Anpassung**: Der Host kann unbegrenzt Wasser/Seife aktivieren.
  </Card>
  <Card title="Herausforderungs-Modus">
    - **Kompetitiver Koop**: Wettlauf gegen die Zeit.
    - **Spielerlimit**: Streng auf 2 Spieler begrenzt.
    - **Belohnungen**: Exklusive kosmetische Abzeichen für Goldmedaillen.
  </Card>
</CardGrid>

Bei der Nutzung von **PowerWash Simulator 2 Crossplay** fungiert die Internetverbindung des Hosts als Anker. Wenn der Host eine hohe Latenz hat, sehen Gäste möglicherweise „Geister-Schmutz“, bei dem Objekte für sie sauber erscheinen, auf dem Bildschirm des Hosts jedoch schmutzig bleiben. Um dies zu mildern, sollte immer der Spieler mit der stabilsten Glasfaserverbindung als Sitzungshost fungieren.

| Modus-Funktion | Karriere-Modus | Freies Spiel | Herausforderungs-Modus |
| :--- | :--- | :--- | :--- |

| **Verdiente Währung** | Voll (Host) / Teilweise (Gast) | Keine | Keine |
| **Crossplay Aktiv** | Ja | Ja | Ja |

Teilen Sie die Karte in Quadranten auf. Weisen Sie einen Spieler dem Dach zu, zwei den Wänden und einen dem Boden/Detailing. Dies verhindert ein „Überkreuzen der Düsen“ und stellt sicher, dass keine Stellen übersehen werden, was Ihren Stundenverdienst maximiert.

<Section icon={Shield} title="Fehlerbehebung bei Crossplay-Verbindungsproblemen">

Obwohl das **PowerWash Simulator 2 Crossplay**-System robust ist, können Spieler gelegentlich auf Verbindungshürden stoßen. Diese hängen oft mit Netzwerksicherheitseinstellungen oder plattformspezifischen Einschränkungen zusammen. Im Jahr 2026 haben die meisten Konsolen strikte NAT-Typen, die die Peer-to-Peer-Natur der Multiplayer-Architektur des Spiels stören können.

Wenn Sie beim Versuch, einem Freund beizutreten, die Fehlermeldung „Verbindung abgelaufen“ erhalten, ist die häufigste Lösung, den Schalter „Online-Multiplayer“ aus- und wieder einzuschalten. Manchmal benötigt der Handshake zwischen dem Backend des Spiels und dem Dienst der Plattform (wie PlayStation Network oder Xbox Live) eine Aktualisierung.

| Fehlermeldung | Wahrscheinliche Ursache | Empfohlene Behebung |
| :--- | :--- | :--- |
| **Raum nicht gefunden** | Abgelaufener Code / Falsche Version | Neuen Code generieren / Spiel aktualisieren |
| **NAT inkompatibel** | Router-Firewall | UPnP aktivieren oder Portweiterleitung |
| **Sitzung voll** | Spielerlimit erreicht | Modus-Limits prüfen (2 oder 6) |
| **Verbindung verloren** | Host getrennt | Sitzung neu hosten |

<Callout type="success" title="Erfolgs-Tipp">
Die Verwendung einer kabelgebundenen Ethernet-Verbindung anstelle von WLAN reduziert „Desync“-Probleme erheblich, bei denen der Wasserstrahl während intensiver **PowerWash Simulator 2 Crossplay**-Sitzungen hinter Ihren Bewegungen herzulaufen scheint.
</Callout>

<Section icon={Trophy} title="Gemeinsamer Fortschritt und Gast-Belohnungen">

Eine häufige Frage zu **PowerWash Simulator 2 Crossplay** betrifft die Handhabung von Belohnungen und Fortschritt für diejenigen, die nicht hosten. Die Entwickler haben für 2026 ein „Söldner-System“ implementiert, das sicherstellt, dass Gäste für ihre Arbeit entschädigt werden, auch wenn der Missionsfortschritt nur für die Karriere des Hosts zählt.

Gäste verdienen einen Prozentsatz des Gesamtwerts des Auftrags, der direkt auf ihr persönliches Konto eingezahlt wird. Dieses Geld kann verwendet werden, um neue Hochdruckreiniger, Düsen und Reinigungsflüssigkeiten in ihren eigenen Einzelspieler- oder gehosteten Spielen zu kaufen. Kosmetische Freischaltungen, die an bestimmte Missionsabschlüsse gebunden sind (wie die „Goldene Düse“), müssen jedoch im eigenen Karriere-Speicherstand verdient werden.

| Belohnungstyp | Host-Vorteil | Gast-Vorteil |
| :--- | :--- | :--- |
| **Missionsfortschritt** | 100 % gespeichert | 0 % gespeichert |
| **Basiswährung** | 100 % Auszahlung | 25-50 % Auszahlung |
| **EP / Leveln** | Voller Fortschritt | Voller Fortschritt |
| **Erfolge** | Die meisten freigeschaltet | Ausgewählte freigeschaltet |

<Callout type="info" title="Ausrüstungs-Sync">

</Callout>
