# Können Sie PowerWash Simulator 2 cross-platform spielen?

## 📌 Schritt-für-Schritt-Anleitung zum cross-platformen Spiel

### Schritt 1: Verknüpfen Sie Ihr Square Enix-Konto

Um cross-platformes Spiel zu aktivieren, müssen Sie Ihr Square Enix-Konto verknüpfen. Hier ist, wie Sie es tun:

1. Starten Sie PowerWash Simulator 2 und gehen Sie zum Hauptmenü.
2. Wählen Sie die Option, sich anzumelden oder ein Square Enix-Konto zu erstellen.
3. Folgen Sie den Anweisungen auf dem Bildschirm, um Ihr Konto zu verknüpfen.

### Schritt 2: Fügen Sie einen Freund für cross-platformes Spiel hinzu

Sobald Ihr Square Enix-Konto verknüpft ist, können Sie Freunde zu Ihrem Spiel hinzufügen. Hier ist, wie Sie einen Freund für cross-platformes Spiel hinzufügen:

1. Gehen Sie zum Koop-Menü in PowerWash Simulator 2.
2. Wählen Sie "Freund hinzufügen" und geben Sie die Square Enix-ID Ihres Freundes ein.
3. Stellen Sie sicher, dass Ihr Freund auch in seinem Square Enix-Konto angemeldet ist.

### Schritt 3: Treten Sie oder hosten Sie ein cross-platformes Spiel bei

Nun, da Sie Freunde hinzugefügt haben, können Sie ein cross-platformes Spiel beitreten oder hosten. Hier ist, wie Sie es tun:

1. Gehen Sie zum Hauptmenü in PowerWash Simulator 2.
2. Wählen Sie "Spiel beitreten" oder "Spiel hosten" je nach Ihrem Vorlieben.
3. Wählen Sie das Spielmodus aus, den Sie spielen möchten, z.B. "Karriere-Modus" oder "Freies Spiel".
4. Sobald Ihre Freunde die Einladung akzeptieren, sind Sie alle im selben Spiel, egal auf welcher Plattform.
